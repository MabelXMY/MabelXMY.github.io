走过2019总结-台湾交换经历之柚子铺

发表于 2020-03-05 更新于 2020-03-06 分类于 总结

2019年在台湾交换的期间我加入了「柚子铺」的微信运营小团队（柚子铺是最大的台湾交换生服务平台，致力为交换生提供交换期间吃喝玩乐学的全方位服务，致力于让陆台有志年轻人一起创造不一样的交换经历），现在想来是我觉得自己做的最正确的决定之一。当时面试的时候有一个问题是：

试想你离开柚子铺的时候，你觉得自己的收获会是什么?

我当时的答案是：

因为作为一个研究生老学姐，我实在是太宅了呀，好不容易鼓起勇气来了台湾交换，当然想要认识更多的人，看遍更多的风景，体验更多有趣的事，加入柚子铺，就是这一切动力的源泉。

我的专业是新闻传播学，以前做过相关方面的实习，有一定的经验也有很多的想法，自己也一直在尝试用公众号和视频等形式记录和分享自己的生活和感想，所以如果能加入柚子铺这个组织，和一群人一起做这件事，我们会有更大的能量，做出我一个可能无法企及的成果，所以我很期待。

期待我的收获：认识一群志同道合的小伙伴并且经由这一份工作得以获得一份不一样的在台湾的体验，走出自己整日窝在宿舍不愿出门的舒适圈，同时锻炼了自己的专业能力，这些都是我期待的收获。

现在想来，我所期待的应该都得到了，我在柚子铺的微信公众号也产出了几篇文章，做年终总结的时候看到有白纸黑字确凿无疑的成果保存，真的是一件让人感到有成就感的事。现在把自己的几篇文章整理一下，放在下面：
